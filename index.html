What is GhostBSD?

A FreeBSD base

Being based on FreeBSD, it ensures excellent performance, reliability and security. There is also a contrast in freedom compared to Linux; BSD having a very permissive license.

Elegant design

Our MATE desktop environment provides an elegant and refined user experience, though other desktop environments are available through the Software Station.

30.000 packages

While it comes with the standard applications of a web browser, LibreOffice suite, etc, there are 30.000 more software packages available.

GhostBSD 22.01.12 ISO is now available

This new ISO contains fixes, improvements, and software updates. Finally, the installer hanging at the cleaning stage for ZFS installation got fixed, and OpenRC and dhcpcd were removed from the base code. Furthermore, automation configuration for HD 7000 series and older GPUs has been added.
